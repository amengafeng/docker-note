查看指定进程

ps -ef | grep docker

查看系统版本

cat /etc/*release*

查看docker版本

docker version

搜索镜像-搜索有关centos的镜像

docker search centos

查看已有的镜像

docker images

下载镜像-下载centos的镜像

docker pull centos

上传镜像-上传centos的镜像

docker push centos

查看默认连接

docker-machine ls

删除连接

docker-machine rm default

修改加速器

#创建一台安装有Docker环境的Linux虚拟机，指定机器名称为default，同时配置Docker加速器地址。
docker-machine create --engine-registry-mirror=https://lag1szk4.mirror.aliyuncs.com -d virtualbox default
#查看机器的环境配置，并配置到本地。然后通过Docker客户端访问Docker服务。
docker-machine env default 
eval "$(docker-machine env default)" 
docker info
直接修改配置文件也可以修改加速器：vim /etc/default/docker

运行容器，并输出“Holle”

docker run centos /bin/echo 'hello'

查看正在运行的镜像

docker ps/docker ps --all

查看docker的基础信息

docker info

构建镜像

docker commit 通过容器构建
			-a 作者信息
			-m 镜像信息
			-p --pause=true
			用例：docker commit -a ’作者名‘ -m ‘镜像信息’ 本地镜像名 上传镜像仓库名称（作者名/镜像名）
		
docker build 通过dockerfile 文件构建
